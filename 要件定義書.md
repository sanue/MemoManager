# メモ管理システム 要件定義書 

## 1. システム概要
本システムは、ユーザーが簡単なメモ（タイトル＋本文）を登録・編集・削除できるWebアプリケーションである。

## 2. 目的
- JavaとSpring Bootを使って、シンプルなCRUD機能を実装する。
- 要件定義から実装までの開発プロセスを体験する。
- 面接で成果物として説明できるプロジェクトを作成する。

## 3. 対象ユーザー
- 開発者本人（ログイン機能なし）

## 4. 機能一覧

| No. | 機能名         | 機能内容                                      |
|-----|----------------|-----------------------------------------------|
| 1   | メモ一覧表示   | 登録された全メモを表示                       |
| 2   | メモ新規登録   | タイトル・本文を入力して新規メモを作成       |
| 3   | メモ編集       | 既存メモの編集                               |
| 4   | メモ削除       | 任意のメモを削除                             |

## 5. 使用技術（予定）
- Java 11
- React
- MySQL

## 6. 制約事項
- ユーザー認証なし
- デザイン最小限
- ユーザーは1人のみ（自分）

## 7. 納期
- 2025年5月2日
